# Release Notes

Все значимые изменения в проекте HLR Bulk Checker документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

---

## [1.6.0] - 2025-01-21

### Добавлено
- Отображение числовых GSM кодов в отчёте (например, "Bad Number (1)")
- Полная таблица GSM кодов в Help Center с описаниями и рекомендациями
- Автоматическое определение invalid по GSM кодам 1, 5, 9, 12

### Изменено
- Колонка "GSM Статус" теперь показывает статус + числовой код
- Номера с Bad Number/Blocked автоматически помечаются как invalid

### Удалено
- Техническая документация из Help Center (API, администрирование)
- Избыточные разделы справки — оставлено только руководство пользователя

### Исправлено
- Логика определения валидности во всех операциях (single/batch/resume)

---

## [1.5.0] - 2025-01-20

### Добавлено
- Новый дизайн Dashboard с улучшенной навигацией
- Help Center с руководством пользователя
- Фильтры по качеству и статусу номеров
- Сортировка по всем колонкам таблицы результатов
- Полноэкранный режим просмотра результатов
- Пагинация результатов (50/100/200 на страницу)

### Изменено
- Улучшен интерфейс таблицы результатов
- Оптимизирована производительность при большом количестве данных

---

## [1.4.0] - 2025-01-19

### Добавлено
- Обработка до 2000+ номеров без ограничений
- Возобновление прерванных проверок
- Прогресс-бар в реальном времени
- Автосохранение результатов каждые 10 номеров

### Исправлено
- Проблема с таймаутом при большом количестве номеров
- Потеря данных при разрыве соединения

---

## [1.3.0] - 2025-01-18

### Добавлено
- Поддержка русского языка
- Поддержка украинского языка
- Переключатель языка в шапке сайта
- Локализация всех сообщений об ошибках

### Изменено

- Форматирование дат согласно выбранной локали

## [1.2.0] - 2025-01-17

### Добавлено
- Health Score (0-100) — оценка качества номера
- Обнаружение дубликатов в загружаемых файлах
- Калькулятор стоимости проверки
- Кэширование результатов на 24 часа
- Индикатор использования кэша

### Изменено
- Улучшен алгоритм нормализации номеров
- Оптимизирован расход API запросов

---

## [1.1.0] - 2025-01-16

### Добавлено
- Собственная система авторизации (логин/пароль)
- Блокировка аккаунта после 5 неудачных попыток входа
- Лимиты проверок на пользователя
- Админ-панель управления пользователями
- Журнал действий пользователей
- Роли: администратор и пользователь

### Безопасность
- Хэширование паролей с bcrypt
- JWT токены для сессий
- Защита от брутфорса

## [1.0.0] - 2025-01-15

### Добавлено
- Массовая проверка номеров через Seven.io API
- Загрузка CSV и TXT файлов
- Экспорт результатов в CSV и Excel

- Тёмная тема интерфейса

---

## Типы изменений

- **Добавлено** — для новых функций
- **Изменено** — для изменений в существующей функциональности
- **Устарело** — для функций, которые будут удалены в будущих версиях
- **Удалено** — для удалённых функций
- **Исправлено** — для исправления ошибок
- **Безопасность** — для исправлений уязвимостей
